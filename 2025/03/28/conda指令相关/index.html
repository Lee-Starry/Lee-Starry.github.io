常用命令

conda指令相关

这篇主要存储conda相关指令

管理conda

检查conda版本

conda --version

获取版本号

conda --version或conda -V

列出所有的环境

conda env list

查看conda下的包

conda list

查看环境管理的全部命令帮助

conda env -h

conda升级

  • 升级conda conda update conda
  • 升级anaconda前要先升级conda conda update anaconda
  • 升级所有包 conda update --all

清理空间

  • 删除没用的包 conda clean -p
  • 删除保存下来的压缩文件 conda clean -t

管理环境

创建环境

  • conda create -n env-name
  • conda create -n env-name python=3.8

激活环境

  • linux source activate env-name
  • windows activate env-name

退出环境

  • linux conda source deactivate
  • windows conda deactivate

复制环境

通过克隆env-name创建一个称为env-name-1的副本
conda create -n env-name-1 --clone env-name

删除环境

conda env remove -n env-name

管理包

安装包

  • conda install package-name
  • conda install package-name==version

卸载包

conda remove package-name

更新包

  • 更新一个包 conda update package-name
  • 更新所有包 conda update --all
