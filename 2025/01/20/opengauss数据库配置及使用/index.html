opengauss数据库配置及使用

版本信息

openEuler20.03LTS
openGauss6.0.0

安装VirtualBox软件

这一步其实没有什么特别的，直接官网找安装包安装就好。其实更推荐使用VMWare，在网卡方面出错的概率会比较小。

在VirtualBox中安装openEuler操作系统

首先在VirtualBox上创建一个新的虚拟机，一些初始设置如图所示，其中的虚拟光盘从官网下载

在虚拟化空间的设置上可以参考官网给出的要求。课程资源中建议的虚拟空间较小，在后续配置中很容易出现空间不足无法更新系统或下载软件的情况

之后进行对虚拟机进行一些新的设置就可以进行使用了。

最后启动虚拟机进入openEuler操作系统的安装界面，按照实验文档进行处理即可。也可以按照官网中的介绍进行选择。

openEuler官网提供了多种可视化界面，可以在安装完openEuler后进行可视化界面的安装。这里我选择了DDE(说实话感觉并不足够稳定)。

  1. 更新软件源sudo dnf update
  2. 安装DDEsudo dnf install dde
  3. 设置以图形界面的方式启动sudo systemctl set-default graphical.target
  4. 重启sudo reboot

DDE的安装需要的空间比较大，如果在前期预先配置的内存不够的话，在这一步会出现问题，可以使用相关命令进行内存空间的重新配置和扩展，但是我选择重新建一个虚拟机。

安装Opengauss并完成客户端访问配置

官方提供在openEuler（openEuler版本大于等于22.03 LTS）操作系统上，通过yum命令一键安装openGauss数据库：yum install opengauss -y，出现Complete!即为安装成功。

使用su - opengauss可以切换到数据库安装过程中创建的初始用户，使用数据库提供的工具命令gsql -d postgres -p 7654 -r即可成功连接到初始数据库postgres

接下来可以使用CREATE和GRANT命令进行用户的创建和设置为系统管理员。

如果主机需要远程连接数据库，必须在数据库系统的配置文件中增加此主机的信息，并且进行客户端接入认证。

可以通过以下命令允许所有IP地址的客户端以Starry用户连接到本机
gs_guc set -N all -I all -h "host all Starry 0.0.0.0/0 sha256"
之后会在数据库配置文件pg_hba.conf中多出现最后一行
